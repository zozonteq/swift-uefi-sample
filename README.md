# swift-uefi-sample
Build UEFI Application with Swift-Embedded.  
`x86_64` and `Aarch64` are supported now.

# Build and Run
```shell
make run # run uefi application(x86_64)
make run ARCH=arm64 # run uefi application(Aarch64)
```
